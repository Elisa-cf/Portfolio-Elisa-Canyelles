General info:

The HP Characters App is a React front end application that makes requests to the Harry Potter API from the users web browser.

In this project I've used REACT and the Harry Potter API, to create an application with a search bar that filters and displays the character info that the user type in. I selected Netlify as a deployment method.

This project was written using:

  • React Hooks
  • React UseEffect
  • JSX
  • Async/Await Axios
  • Rest API

Birthday Reminder App

Congratulate your loved ones

General info:

The Birthday Reminder App is a front-end React application with a local database created with Node.js Expres.js and mySQL.

In this application you can add the birthday of your friends. Add their names, years and an avatar or a picture.

Do you have a special friend? Add his or her birthday as a important by clicking on the checkbox.

You can also double click on one of the birthdays that you have already added on the list, and highlight it as important.

Do you need to delete the full list? Click on the button 'Clear All' or delete one by one by clicking on the cross.

Front End toolbox:

  • React Hooks
  • React UseEffect
  • Styled Components
  • Axios

Back End toolbox:

  • Node.js
  • Express.js
  • mySQL
  • Joi for data validation

Deployment method:

I selected Netlify as a deployment method.

Work in progress:

Creating a local database was a really good practise to understand some basic concepts about the back-end of an application. But the real idea is to put the local database into a cloud. This way the application will be available to everyone without having to install the backend locally.

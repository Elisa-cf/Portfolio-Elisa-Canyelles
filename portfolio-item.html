Film & Serie Hero

Find out where to watch

Film Hero is a React front end application that makes requests to the Movie Database API from the users web browser.

Film Hero is a web application which enables users to find out where to stream, rent or buy a particular film or TV show based on their country. It also displays a carousel of popular movies on the home page and similar movies when the user browses the information for a film.

All the data is provided by the Movie DB.

The creation of FilmHero was done by myself, Elisa Canyelles, and my team Eleonora Weber and Tim Coggins.

This project was written using:

  • React Hooks
  • React Router
  • Styled Components
  • Axios
  • Atomic Design
  • Rest API

We used Agile Methodologies as a project management, in particular SCRUM, we worked following Git workflow and using Jira. We selected Netlify as a deployment method.
